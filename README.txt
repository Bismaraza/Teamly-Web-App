FlowSpace - Productivity Operating System

Tech Stack:
HTML, CSS, JavaScript, jQuery, Basic React, PHP, MySQL, XAMPP

How to Run:
1. Extract FlowSpace_ClickUp.zip
2. Copy folder to C:/xampp/htdocs/
3. Start Apache and MySQL in XAMPP
4. Open phpMyAdmin: http://localhost/phpmyadmin
5. Import database/flowspace.sql
6. Open: http://localhost/FlowSpace_ClickUp/index.html

Note: Admin password hash is included for demo. If it does not work, create a new account from signup page.

Viva Explanation:
FlowSpace is a productivity operating system where users can manage tasks, notes, goals and weekly schedules from a single dashboard. It uses HTML and CSS for UI, JavaScript and jQuery for interactions, basic React for dashboard widgets, PHP for backend processing, and MySQL for data storage.
