# Teamly – Team Collaboration Web Application

## Overview

Teamly is a web-based team collaboration application developed to help users organize tasks, manage notes, and work more efficiently through a clean and user-friendly interface.

## Features

* User Registration & Login
* Dashboard Interface
* Task Management
* Notes Management
* Responsive Design
* Multi-Page Navigation

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* PHP

### Database

* MySQL

## Installation & Usage

### Prerequisites

* XAMPP
* PHP
* MySQL

### Steps to Run

1. Clone the repository:

```bash
git clone https://github.com/Bismaraza/Teamly-Web-App.git
```

2. Move the project folder to:

```text
xampp/htdocs/
```

3. Start Apache and MySQL from XAMPP.

4. Import the database.

5. Open:

```text
http://localhost/Teamly-Web-App
```
